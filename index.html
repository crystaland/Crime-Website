  • About the App

    CrimeMap is an app that allows college students to view the crime that is occurring in their university communities. Students have the option to view crimes, report crimes, and call for help through the app.

    What we used to make it

    Our team used HTML, CSS, GeoJSONs, the Mapbox API, and Javascript to create the website. For the app, we used the Google Maps API and Swift. The app prototype is being run through Proto.io, and the website is currently hosted by Github.
